# Triage ラベル (Triage Labels)

skills は 5 つの canonical triage roles で語る。このファイルはそれらの roles を、この repo の issue tracker で実際に使う label 文字列にマッピングする。

| mattpocock/skills でのラベル | 自分のトラッカーでのラベル | 意味                                  |
| -------------------------- | -------------------- | ---------------------------------------- |
| `needs-triage`             | `needs-triage`       | Maintainer がこの issue を評価する必要がある  |
| `needs-info`               | `needs-info`         | 報告者からの追加情報待ち |
| `ready-for-agent`          | `ready-for-agent`    | 仕様が完全に固まっており、AFK agent の準備ができている  |
| `ready-for-human`          | `ready-for-human`    | 人間による実装が必要            |
| `wontfix`                  | `wontfix`            | 対応しない                     |

skill が role に言及するとき（例: "apply the AFK-ready triage label"）、この表の対応する label 文字列を使う。

右列を実際に使う語彙に合わせて編集する。
