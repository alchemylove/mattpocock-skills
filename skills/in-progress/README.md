# 開発中 (In Progress)

まだ開発中の skills。ship 準備ができていない — rough edges、breaking changes、abandoned experiments を expect すること。stable bucket に graduate するまで plugin とトップレベル README から除外される。

- **[review](./review/SKILL.md)** — 固定 point 以降の changes を 2 つの parallel axis で review する: **Standards**（diff は repo の coding standards に従っているか？）と **Spec**（diff は originating issue/PRD を faithfully に implement しているか？）。
- **[writing-beats](./writing-beats/SKILL.md)** — choose-your-own-adventure 風に、article を beats の journey として shape する。starting beat を選び、その beat だけ write し、natural end に達するまで next に pivot する。
- **[writing-fragments](./writing-fragments/SKILL.md)** — heterogeneous nuggets of writing — fragments — を掘り起こす grilling session。将来の article 向け raw material として 1 document に append する。
- **[writing-shape](./writing-shape/SKILL.md)** — raw material の markdown file を取り、各 step で format choices を argue しながら paragraph ごとに article に shape する。
